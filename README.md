# sprite picker Chrome extension

1. **Load the extension**: `chrome://extensions/` -> Enable Developer Mode -> Load Unpacked -> select `./tools/sprite-picker/`
2. **Start level editor**: `vite` then open `http://localhost:5173/level-editor.html`
3. **Browse sprites**: Navigate to `spriters-resource.com` -OR- `https://easierbycode.com/spriteX`, click the green **+** button (bottom-right) to enter pick mode
4. **Detect**: Click any sprite sheet image -- red bounding boxes appear around detected sprites
5. **Select**: Click sprites to toggle selection (green = selected)
6. **Send**: Choose atlas + mode in the panel, click "Send to Level Editor" -- the Atlas Manager opens with your sprites
